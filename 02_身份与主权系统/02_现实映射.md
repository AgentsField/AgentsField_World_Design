```
─────────────────────────────────────────

    signer_public_key,
    canonicalize_json_without_sig(request),
    request["signature"]
  )

    reject_request("INVALID_SIGNATURE")
    return

    return
```

### Shamir秘密共享技术

```
社交备份技术实现：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

场景：将私钥分成5份，需要至少3份才能恢复

分割流程：
1. 将私钥编码为多项式系数
   secret = private_key_bytes
   
2. 创建多项式
   f(x) = secret + a1*x + a2*x^2 (在有限域上)
   where a1, a2 是随机系数

3. 生成5个分片
   shares = []
   for i in 1..5:
     share = (i, f(i))
     shares.append(share)

4. 分发分片给5个信任星球
   send_share(trusted_planet_1, shares[0])
   send_share(trusted_planet_2, shares[1])
   ...

恢复流程：
1. 收集至少3个分片
   received_shares = [
     (1, f(1)),
     (3, f(3)),
     (5, f(5))
   ]

2. 使用拉格朗日插值恢复秘密
   secret = lagrange_interpolate(received_shares)

3. 验证恢复的私钥正确性
   assert compute_opid(secret) == my_opid

安全特性：
✓ 任何2个分片无法恢复私钥
✓ 任何3个分片可以恢复私钥
✓ 即使部分盟友背叛，私钥仍然安全

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
