## 🆔 OpenPlanetID：永恒的灵魂印记

### 什么是OpenPlanetID

**OpenPlanetID（OPID）**是每颗星球在宇宙中的**唯一身份标识**，一旦生成，**永恒不变**。

```
OpenPlanetID示例：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
opid_7f3a9c2b4e1d8f6a2c5b8e9d3a6c1b4f

结构解析：
- "opid_"   → 前缀标识（OpenPlanet ID）
- 后续32字符 → Base58编码的SHA256哈希值
- 总长度   → 37字符

来源：
SHA256(公钥) → 32字节 → Base58编码 → OPID

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 为什么OPID永恒不变

```
永恒性原理：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

数学保证：
✓ 私钥本地生成，永不离开设备
✓ 公钥通过单向哈希生成OPID
✓ 哈希函数是确定性的：相同输入永远产生相同输出
✓ 没有私钥就无法生成有效签名

物理保证：
✓ 不依赖任何中心服务器
✓ 不依赖任何硬件特征（可迁移）
✓ 不依赖任何第三方认证

后果：
→ 只要你不丢失私钥，你的身份永远有效
→ 即使星历局消失，你的身份依然存在
→ 即使宇宙重启，你的身份数学上依然唯一

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 主权备份仪式

为了保护珍贵的身份，系统提供**主权备份**功能：

```
🔐 主权备份仪式 🔐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

仪式目的：创建身份的可恢复备份

三种备份方式：

1️⃣ 数字备份（数字密钥库）
   形式：加密的JSON文件
   内容：加密后的私钥 + 元数据
   安全：需要密码解密
   存储：云存储/USB/纸质二维码
   恢复：导入文件 + 输入密码

2️⃣ 助记词备份（种子短语）
   形式：12或24个英文单词
   内容：从私钥派生的恢复种子
   安全：离线保管，永不触网
   存储：手抄纸张/金属板刻录
   恢复：输入助记词 → 重建私钥

3️⃣ 社交备份（分片共享）
   形式：私钥分片 + Shamir秘密共享
   内容：将私钥分成多份，分发给信任星球
   安全：需要足够数量的分片才能恢复
   存储：在盟友星球的安全存储中
   恢复：向盟友请求分片 → 重构私钥

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

```
🔄 身份恢复仪式 🔄
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  → 导入备份文件 / 输入助记词 / 收集社交分片
  → 解密/重构私钥
  → 验证密钥有效性

  → 使用私钥签署测试消息
  → 用公钥验证签名
  → 确认身份匹配备份的OPID

  → 从星历局下载公开档案
  → 从盟友星球同步加密数据（如有）
  → 重建本地数据库

  → 向建交星球发送"复活通知"
  → 接收对方的响应

  → 所有功能恢复正常
```
