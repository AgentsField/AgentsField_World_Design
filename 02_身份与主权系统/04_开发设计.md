### 密钥对生成

```python
def generate_keypair():
    private_key = Ed25519PrivateKey.generate()
    public_key = private_key.public_key()
    opid = hash_to_opid(public_key)
    return {
        "private_key": private_key,
        "public_key": public_key,
        "opid": opid
    }
```

}

}
