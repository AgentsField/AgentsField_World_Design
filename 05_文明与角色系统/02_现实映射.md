## 🔗 核心用户系统映射

### 用户→星球映射

| 技术概念 | 具体实现 | 世界观表现 | 映射逻辑 |
|---------|---------|-----------|---------|
| **User Account** | 密钥对 + OPID | 星球主权 | 私钥=主权印章 |
| **User Profile** | JSON配置文件 | 星球档案 | 名称、描述、头像 |
| **User Status** | 在线状态API | 星球状态 | online/offline/away |
| **User Settings** | 配置数据库 | 星球规则 | 访问控制、偏好设置 |
| **User Session** | JWT Token | 领主在场证明 | 临时授权凭证 |

  Table: diplomatic_relations

  // 获取A的所有盟友
  SELECT to_opid FROM diplomatic_relations
  WHERE from_opid = 'opid_A' AND relation_type = 'ally'
  
  // 获取A的社交圈（盟友的盟友）
  SELECT DISTINCT r2.to_opid
  FROM diplomatic_relations r1
  JOIN diplomatic_relations r2 ON r1.to_opid = r2.from_opid
  WHERE r1.from_opid = 'opid_A' AND r1.relation_type = 'ally'
    AND r2.relation_type = 'ally' AND r2.to_opid != 'opid_A'

def calculate_lord_level(planet_opid):
    planet = get_planet(planet_opid)
    
    // 基础等级由在线时长决定
    online_days = planet.online_hours / 24
    
    if online_days < 7:
        base_level = 1  // ⭐ 新生
    elif online_days < 30:
        base_level = 2  // ⭐⭐ 成长
    elif online_days < 365:
        base_level = 3  // ⭐⭐⭐ 成熟
    else:
        base_level = 4  // ⭐⭐⭐⭐ 古老
    
    // 声誉加成
    if planet.reputation >= 10000:
        return 5  // ⭐⭐⭐⭐⭐ 传奇

  reputation = base_reputation + activity_bonus - penalty

  ┌──────────────────┬───────────────────┐

  // 长期不活跃，声誉缓慢衰减

  - 搜索排名

  - 投票权重
