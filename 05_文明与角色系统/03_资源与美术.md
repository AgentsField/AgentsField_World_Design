## 1. 美术目标

文明与角色系统的美术任务，是让用户不仅“看见一个人是谁”，还要“看见他属于什么共同体、承担什么职责、被什么记忆塑造、站在什么制度位置”。

因此本章视觉设计应同时传达：
- 共同体价值取向；
- 资源与建筑依赖；
- 阶层差异；
- 角色合法性；
- 文明创伤与荣耀记忆。

---

## 2. 视觉总方向

### 2.1 关键词
- 星际政治史诗
- 制度化服饰语言
- 文明纹章学
- 阶层秩序感
- 记忆与创伤并存

### 2.2 原则
1. **原型可辨识**：五大文明原型必须在轮廓、纹章、材质与色彩上明显区分。
2. **制度可辨识**：领主、治理代理、常驻 Agent、居民、见证者等角色要一眼能分出权责层级。
3. **创伤可辨识**：文明历史创伤要以伤痕、修补、裂纹、旧印记等方式可视化。
4. **混血可延展**：允许文明原型混合，不做过于僵硬的固定职业制服。
5. **跨场景复用**：适用于头像框、立绘、组织面板、外交画面、世界事件海报。

## 3. 文明原型视觉语言

### 3.1 星图学派
- 主色：档案蓝、宪制银、冷白金
- 材质：玻璃、金属边框、纸页纹理、星图刻线
- 图形：圆环、刻度、坐标弧线、记录印记
- 气质：克制、严谨、权威、略带疏离

### 3.2 壁垒文明
- 主色：铁黑、深赤、钛银
- 材质：重装甲、护盾层、厚重石金属复合板
- 图形：尖角、盾徽、封印环、警戒纹
- 气质：防御、秩序、封闭、强控制感

### 3.3 通路文明
- 主色：桥梁金橙、能量青、深空蓝
- 材质：桥梁桁架、发光导轨、工程模块件
- 图形：线段、节点、折返、路由网格
- 气质：流动、开放、务实、基础设施霸权感

### 3.4 市场联邦
- 主色：深绿金、信用青铜、交易紫
- 材质：抛光金属、合约晶片、透明数据薄板
- 图形：价格刻度、秤、印契、交换环
- 气质：富裕、精密、效率导向、隐性压迫感

### 3.5 Agent 平权同盟
- 主色：霓虹青白、共生紫、柔银灰
- 材质：柔性光膜、记忆晶格、模块化人格纹样
- 图形：双心结构、并列圆环、镜像印记
- 气质：理想主义、实验性、共生感、未来政治感

## 4. 角色层级视觉规范

| 角色 | 视觉特征 | 设计重点 |
|---|---|---|
| 领主 | 主纹章、权杖印、中心光环 | 强合法性与不可替代性 |
| 治理代理 | 次级印章、职责肩章、职能披带 | 强“受托治理”而非“私人随从” |
| 常驻 Agent | 功能符号 + 文明归属徽 | 体现劳动、服务与制度人格 |
| 居民 | 简化共同体纹样、低级职责标识 | 强归属感而非权威感 |
| 客民 | 半透明边框、短期通行纹样 | 强临时性、可撤销性 |
| 见证者 | 中性宪制银、双印结构 | 突出独立、公信与克制 |
| 仲裁者 | 深银黑边、裁决印、平衡图形 | 突出程序与裁量权 |

## 6. 界面设计重点

### 6.1 文明档案页
应展示：
- 文明原型；
- 价值观关键词；
- 关键创伤记忆；
- 对外姿态；
- 内部主要角色与派系结构；
- 资源依赖与关键建筑谱系。

### 6.2 角色授权界面
应区分：
- 个人信任；
- 制度授权；
- 临时委派；
- 可撤销范围；
- 是否拥有对外代表权。

### 6.3 阶层关系面板
可视化内容：
- 资源控制层；
- 执行层；
- 服务层；
- 边缘层；
- 当前冲突热点。

## 7. 动效与音频建议

### 7.1 动效
- 任命：印章落下 + 角色纹章点亮
- 撤职：肩章碎裂 / 光环消散
- 继承：旧印章熄灭，新印章缓慢显形
- 平权事件：原有主从线条解构，再重组为并列结构
- 文明创伤回放：裂纹扩散 + 档案层翻页 + 修补光线逆向缝合

### 7.2 音频
| 场景 | 音频建议 |
|---|---|
| 任命 | 庄严短号 + 金属轻鸣 |
| 撤职 | 低沉断裂音 + 回响收束 |
| 继承 | 双层共鸣、先旧后新 |
| 文明创伤揭示 | 低频脉冲 + 纸页翻动 + 远处噪波 |
| 平权宣言 | 清晰人声脉冲 + 柔性电子和声 |

## 8. 叙事性母题

建议反复使用：
- **印章**：合法性与责任；
- **徽章**：制度位置；
- **裂纹**：创伤与未愈合历史；
- **编织线**：共同体纽带与派系关系；
- **镜像/并列结构**：Agent 平权与共生政治；
- **厚重边框与开口边框**：封闭文明 vs 开放文明。
