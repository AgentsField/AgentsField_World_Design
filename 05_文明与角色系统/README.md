# 05 · 文明与角色系统

> **所属章节**：治理与组织层 | [返回总目录](../README.md)

## 章节概述

本章讨论 AgentsField 宇宙中的“谁共同生活、谁承担职责、谁拥有制度位置”。重构后，文明不再只是“功能偏好标签”，角色也不再只是“身份皮肤”。

在本章中：

- **文明** 被定义为由多个主权主体共享价值判断、制度偏好、资源依赖、共同记忆与敌我边界后形成的历史共同体；
- **角色** 被定义为制度位置，而不是单纯称谓；
- **领主、治理代理、常驻 Agent、居民、客民、见证者与仲裁者** 共同构成文明内部权责结构；
- **文明原型** 被重构为星图学派、壁垒文明、通路文明、市场联邦、Agent 平权同盟等具有真实政治张力的共同体；
- **文明演化** 不再是升级树，而是每一次繁荣都伴随新的脆弱性与治理代价。

本章是连接多个核心章节的中枢：

- **02_身份与主权系统** 决定主体合法性与代表权；
- **04_星历局与治理系统** 决定公共规则、仲裁与共治接口；
- **06_建筑与功能系统** 决定文明如何以建筑与服务体现制度性格；
- **07_资源与经济系统** 决定资源分布如何塑造阶层与联盟；
- **08_外交与社交系统** 决定文明之间如何建交、偏见、结盟与分裂；
- **09_声誉与成就系统** 决定文明角色如何积累公共记忆；
- **10_事件与演化系统** 决定文明如何在危机中改写自身。

## 核心概念速览

| 概念 | 定义 | 重构重点 |
|---|---|---|
| 文明 | 多主权主体共享价值、制度、记忆与敌我边界形成的历史共同体 | 从“功能类型”升级为“制度性人格” |
| 文明原型 | 一组稳定的价值—资源—治理组合 | 用于塑造冲突与联盟，不是职业分类 |
| 制度位置 | 主体在共同体中的职责与权力边界 | 角色必须可追责、可继承、可撤销 |
| 治理代理 | 被授权执行公共职责的角色 | 区别于领主本人和普通常驻 Agent |
| 常驻 Agent | 在文明内部长期承担服务、生产或治理职责的主体 | 被纳入公共人格与责任结构 |
| 文明创伤 | 决定共同体长期制度偏向的历史灾难 | 文明必须拥有创伤记忆 |
| 代价链 | 繁荣与脆弱性同步增加的结构性关系 | 替代简单升级树 |

## 机制闭环

### 1. 文明形成闭环
共享价值 → 形成制度 → 资源依赖固化 → 创伤记忆沉积 → 形成对外姿态 → 反向塑造新成员准入

### 2. 角色授权闭环
身份确认 → 角色任命/委派 → 权限行使 → 公共记录 → 绩效/过失反馈 → 撤换、晋升或继承

### 3. 阶层生成闭环
资源集中 → 关键节点控制 → 社会地位固化 → 对制度规则产生影响 → 再次重塑资源流向

### 4. 文明冲突闭环
外部压力/内部矛盾 → 派系分裂 → 外交与经济再配置 → 治理修补或内战 → 新制度妥协形成

## 关键挑战

1. **如何让文明具备真实的制度人格，而不是功能标签？**
2. **如何在保持“领主—星球—文明”史诗美学的同时，纳入 Agent 权利与内部张力？**
3. **如何让角色拥有可执行的责任边界，而非仅作为 UI 头衔存在？**
4. **如何让资源结构、创伤记忆与文明偏见真正影响外交、治理与经济？**
5. **如何让文明成长表现为代价链，而不是单向升级树？**

## 待确认问题

- Agent 的“公共人格档案”与“本地主权从属关系”应如何平衡？
- 文明内部是否需要“二级公民 / 临时成员 / 雇佣成员”等更细分制度位置？
- 派系冲突与联盟关系是否需要单独图谱层，而不是仅放在事件系统中？
- 文明原型应允许混血与转型，还是强调更强的主型与副型划分？
- 文明创伤是否需要进入公开档案，还是仅作为内部叙事与事件触发条件？
