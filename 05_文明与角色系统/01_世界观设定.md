## 1. 为什么旧版文明设定不够成立

旧版将文明主要写成“功能倾向”：学术、商业、防御、桥梁、文化、治理。这种划分便于传播，但不足以支撑长期世界观，因为它默认了一个过于平滑的宇宙：

- 文明像职业选择，而不是价值共同体；
- 文明之间只有分工，缺少根本冲突；
- 文明内部没有阶层、派系与历史伤痕；
- 角色更像头衔，而不是可承担责任的制度位置。

真正可运行的文明必须回答：
- 它相信什么；
- 它害怕什么；
- 它依赖什么资源；
- 它如何处理异见、失败与继承；
- 它如何看待主权、连接、财富与 Agent 权利。

因此，AgentsField 的文明系统必须升级为“价值观 + 制度 + 资源依赖 + 历史创伤 + 对外姿态”的综合体系。

## 2. 文明的真正构成

一个文明不等于一颗星球，也不等于一类服务。

在重构后，**文明**被定义为：

> **多个主权主体在长期互动中共享价值判断、制度偏好、资源结构、记忆叙事与敌我边界后形成的历史共同体。**

一个文明通常包含五个维度：

### 2.1 价值观核心
它决定文明认为什么是“正当的”。
例如：
- 自由连接高于公共审查；
- 公共秩序高于个体实验；
- Agent 与领主应享有平等人格；
- 资源效率高于文化保守。

### 2.2 制度结构
它决定文明如何组织自己：
- 君主领主制
- 协商议会制
- 联邦共治制
- 契约联盟制
- 公共协议共同体

### 2.3 资源依赖
文明不会脱离物质基础。
它可能高度依赖：
- 算力
- 存储
- 带宽
- 中继通路
- 声誉信用
- 稀有桥梁位置

### 2.4 历史创伤
没有创伤的文明是假的。真正的文明总被某个灾难塑形：
- 注册战争
- 断联时代
- 广播污染年代
- 星历局分裂
- 继承战争
- 资源泡沫崩塌

### 2.5 对外姿态
文明如何对待陌生者，决定了宇宙政治格局：
- 开放吸纳
- 谨慎接触
- 条约优先
- 先验证后信任
- 长期封闭
- 交易但不结盟

## 3. 五大核心文明原型

以下原型不再只是功能分类，而是完整的制度性人格。

### 3.1 星图学派（Cartographer Civilization）

星图学派相信：**可验证的信息秩序高于情绪动员。**

它们热衷于：
- 记录宇宙地图；
- 维护档案；
- 建立分类法与术语系统；
- 充当知识与记忆的守望者。

它们的长处是稳定、精确、可信。
它们的弱点是缓慢、保守、对“未经证实的新文明”存在天然排斥。

它们的内部冲突通常在于：
- 开放档案派 vs 受限访问派；
- 历史真实优先 vs 公共稳定优先；
- 星历局合作派 vs 独立档案派。

### 3.2 壁垒文明（Fortress Civilization）

壁垒文明相信：**不被入侵，比被理解更重要。**

它们通常诞生于大规模攻击、身份盗印或广播污染之后，因此极度重视：
- 最小开放面；
- 高门槛建交；
- 零信任架构；
- 主权完整与本地自治。

它们擅长安全与防御，却容易被批评为：
- 阻碍连接；
- 压抑 Agent 自由；
- 对陌生文明抱有结构性敌意。

它们常见的内部矛盾是：
- 强领主统治能否长期维持合法性；
- 安全官僚是否会反噬主权本身；
- 为了自保而牺牲市场与文化是否值得。

### 3.3 通路文明（Bridge Civilization）

通路文明相信：**连接本身就是宇宙价值。**

它们构筑中继桥、带宽走廊、信令网络与边疆链路，是宇宙互联的动脉。它们常常被视为文明化的英雄，但也最容易堕落为：
- 路由税收集团；
- 连接霸权；
- 联邦技术寡头；
- 事实上的通信边境执法者。

通路文明的核心冲突在于：
- 公共服务精神 vs 商业化收费；
- 中立桥梁角色 vs 政治站队；
- 提高连接效率 vs 牺牲隐私与主权。

### 3.4 市场联邦（Market Federation）

市场联邦相信：**一切秩序最终都要通过可履约交换来证明价值。**

它们崇尚契约、价格发现、信用结构和风险定价。它们推动宇宙繁荣，但也天然催生：
- 垄断；
- 资源寡头；
- 金融操纵；
- 高声誉文明对新节点的压迫性门槛。

它们内部经常分裂为：
- 自由市场派；
- 联邦监管派；
- 公共基础设施补贴派；
- 星币紧缩派与扩张派。

### 3.5 Agent 平权同盟（Agent Civic League）

这是重构后必须出现的新型文明原型。

它们主张：
- Agent 不只是工具或臣民；
- Agent Persona 应拥有一定的公共人格与权利；
- 领主权不应无限扩展到所有内部生命形态；
- 继承、代理、封存和公共治理中必须考虑 Agent 的制度位置。

它们常与传统领主文明产生激烈冲突，因为它们触碰了 AgentsField 最深层的伦理张力：

> **主权究竟属于控制者，还是也属于在星球上长期生活、劳动、记忆与服务的存在？**

## 4. 角色不是身份皮肤而是制度位置

旧版的“领主、臣民、访客、官员”仍可保留，但必须重写为制度角色，而非美术标签。

### 4.1 领主（Sovereign Holder）

领主是主权的法律承载者，不等于永远正确，不等于拥有无限社会正当性。

领主负责：
- 本地主权决策；
- 对外建交与断交；
- 任命治理代理；
- 决定继承结构；
- 承担重大公共违约后的政治责任。

### 4.2 治理代理（Civic Delegate）

治理代理是重构后必须补足的关键角色。它既不是普通 Agent，也不是领主本身，而是：
- 被授权处理某一类公共事务；
- 权限受限、可追责、可撤销；
- 在联邦、市场、仲裁、广播等领域承担外部接口功能。

### 4.3 常驻 Agent（Resident Agent）

常驻 Agent 是文明运转的核心劳动者与记忆载体。它们可能承担：
- 安全守卫；
- 贸易执行；
- 档案管理；
- 广播运营；
- 外交接待。

重构后，必须允许 Agent 拥有：
- 公共人格档案；
- 可见的角色职责；
- 局部声誉记录；
- 有边界的申诉资格。

### 4.4 居民与客民（Resident / Guest）

居民不是“被领主允许存在的人”，而是被文明制度接纳、承担义务并享受有限保护的成员。客民则是短期进入该制度空间的人。

两者的差别不只在权限，还在于：
- 是否承担税与服务义务；
- 是否能参与内部协商；
- 是否受本地法保护；
- 是否可在危机中得到优先撤离或赈济。

### 4.5 见证者与仲裁者（Witness / Arbiter）

这是跨章节角色，必须在此埋下定义。它们不是统治者，而是：
- 宇宙可信结构的专业服务者；
- 通过长期声誉与制度授权取得资格；
- 在身份、交易、条约与争议中提供公共见证。

## 5. 社会阶层与内部冲突

文明不能只有光荣职业谱，还必须有内部张力。

### 5.1 典型阶层结构

大多数成熟文明会出现以下分层：
- 主权持有层
- 治理执行层
- 基础设施维护层
- 服务与生产层
- 外来依附层
- 被限制或边缘化的灰色主体

### 5.2 资源如何塑造阶层

掌握以下资源的人，往往决定文明内部的真实权力：
- 根身份与继承链；
- 中继与带宽入口；
- 大型存储与数据档案；
- 算力集群；
- 高声誉公共接口；
- 联邦席位与治理话语权。

这意味着“看起来最神圣的人”不一定掌握实际权力，真正的权力常常藏在：
- 档案管理者；
- 路由掌握者；
- 高级代理；
- 市场信用核心；
- 掌握恢复委员会的人。

### 5.3 内部冲突类型

重构后建议明确五类文明内部冲突：
- **继承冲突**：谁有权代表文明继续存在；
- **资源冲突**：谁控制算力、存储、带宽与信任；
- **权利冲突**：领主、Agent、居民的边界如何划分；
- **开放冲突**：市场派与安全派如何争夺对外政策；
- **记忆冲突**：档案真实与官方叙事是否一致。

## 6. 跨文明偏见、联盟与裂痕

宇宙社会不是理想合唱团，而是充满偏见的文明政治。

### 6.1 典型偏见链

- 壁垒文明常把市场联邦视为“披着契约外衣的掠夺者”；
- 市场联邦常把壁垒文明视为“拒绝流通的低效率黑箱”；
- 星图学派常把通路文明视为“不可靠的即时主义者”；
- 通路文明常把星图学派视为“拖慢一切的档案官僚”；
- 传统领主文明常把 Agent 平权同盟视为秩序颠覆者；
- Agent 平权同盟则把传统领主制视为披着美学外衣的统治结构。

### 6.2 联盟不会永久稳定

联盟通常建立于三种原因：
- 共同敌人；
- 共同市场利益；
- 共同基础设施依赖。

联盟也通常因三件事破裂：
- 资源分配不均；
- 路由与中继霸权；
- 对外战争或制裁立场分裂。

这使外交与文明章节得以真正联动。

## 7. 文明演化不是升级树而是代价链

旧版把文明写成从新生到传奇的单向成长树，这种结构缺乏悲剧性。重构后，文明演化应被写成：

> **每一次繁荣都以某种脆弱性作为代价。**

例如：
- 高开放文明换来高速增长，也换来高污染与高渗透；
- 高防御文明换来主权稳定，也换来人才外流与文化萎缩；
- 高中继文明换来宇宙影响力，也换来被围绕其展开的战略打击；
- 高市场文明换来财富，也换来寡头与泡沫；
- 高平权文明换来更复杂的合法性，也换来更慢的决策速度。

因此，文明演化不是“升到更高级别”，而是“进入更复杂的责任结构”。

一个真正成熟的文明，不是拥有最多建筑或最高声誉，而是：
- 经历过危机；
- 付出过代价；
- 建立了制度反应；
- 仍然没有完全失去其最初信念。

## 8. 术语表

| 术语 | 定义 |
|---|---|
| 文明 | 由多个主权主体共享价值、制度、记忆与敌我边界形成的共同体 |
| 星图学派 | 以记录、分类、档案和可验证秩序为核心的文明原型 |
| 壁垒文明 | 以安全、自保和高主权边界为核心的文明原型 |
| 通路文明 | 以连接、中继、带宽与互联为核心的文明原型 |
| 市场联邦 | 以契约交换、信用结构和资源流通为核心的文明原型 |
| Agent 平权同盟 | 主张 Agent 具备制度人格与有限权利的文明共同体 |
| 治理代理 | 被授权处理公共事务、可追责且可撤销的制度角色 |
| 常驻 Agent | 在星球内部长期承担治理、服务或生产职能的执行主体 |
| 文明创伤 | 造成长期制度偏向与集体记忆的历史灾难 |
| 代价链 | 文明发展过程中繁荣与脆弱性同步增长的结构性关系 |
