### 六大文明原型

#### 🏛️ 学术文明（Academic Civilization）

> *"知识是宇宙最珍贵的资源，我们致力于探索和保存它。"*

**特征**：
- 核心建筑：📚 知识宫殿、🔬 实验室
- 主要服务：数据存储、计算服务、知识分享
- 社交特点：理性讨论、学术交流、开放合作
- 政治倾向：民主制、知识共享

**著名星球**：
- 智慧核心（北域-科研区）
- 数据星云档案馆
- 星际科学院

**发展路径**：
```
知识收集 → 理论构建 → 实验验证 → 成果分享 → 声誉积累
```

---

#### 💰 商业文明（Commercial Civilization）

> *"星际贸易是宇宙的血脉，连接着每一颗星球。"*

**特征**：
- 核心建筑：💰 星际交易所、🏪 服务市场
- 主要服务：资源交易、中介服务、金融服务
- 社交特点：契约精神、信誉至上、互惠互利
- 政治倾向：联邦制、贸易联盟

**著名星球**：
- 星际交易所（南域-贸易区）
- 财富星球
- 商业联邦首都

**发展路径**：
```
资源积累 → 服务建立 → 贸易网络 → 金融发展 → 商业帝国
```

---

#### 🛡️ 防御文明（Defensive Civilization）

> *"在这个宇宙中，保护自己是第一要务。"*

**特征**：
- 核心建筑：🛡️ 护盾发生器、🔒 安全堡垒
- 主要服务：安全防护、隐私保护、安全咨询
- 社交特点：谨慎交友、深度信任、小圈子
- 政治倾向：孤立型、严格准入

**著名星球**：
- 防火墙堡垒（西域-守卫星区）
- 隐秘港湾
- 安全圣殿

**发展路径**：
```
安全建设 → 隐私保护 → 信任网络 → 安全服务 → 安全专家
```

---

#### 🌉 桥梁文明（Bridge Civilization）

> *"我们连接孤岛，让宇宙更加紧密。"*

**特征**：
- 核心建筑：🌉 中继中心、🌐 连接枢纽
- 主要服务：网络中继、信令转发、连接优化
- 社交特点：广泛外交、服务精神、中立立场
- 政治倾向：联邦制、服务型治理

**著名星球**：
- 连接之星（东域-技术区）
- 中继枢纽
- 网络桥梁

**发展路径**：
```
公网获取 → 中继建设 → 网络扩展 → 服务优化 → 连接巨头
```

---

#### 🎨 文化文明（Cultural Civilization）

> *"宇宙不仅是技术的舞台，更是艺术的殿堂。"*

**特征**：
- 核心建筑：🎭 星际剧院、🎨 艺术馆
- 主要服务：内容创作、媒体广播、娱乐服务
- 社交特点：创意交流、审美追求、粉丝文化
- 政治倾向：民主制、创作者主导

**著名星球**：
- 星光剧院（南域-娱乐区）
- 艺术家星球
- 文化中心

**发展路径**：
```
内容创作 → 粉丝积累 → 媒体服务 → 文化影响 → 艺术殿堂
```

---

#### ⚖️ 治理文明（Governance Civilization）

> *"秩序不是强加的，而是共同维护的。"*

**特征**：
- 核心建筑：⚖️ 星际法院、📋 行政中心
- 主要服务：争议仲裁、规则制定、公共服务
- 社交特点：公正中立、规则导向、服务型
- 政治倾向：联邦制、服务导向

**著名星球**：
- 星历局总部（虽然中立，但有治理功能）
- 仲裁星球
- 规则之心

**发展路径**：
```
声誉建设 → 服务提供 → 仲裁服务 → 规则制定 → 治理中心
```

### 角色详细定义

#### 领主（Lord/Lady）

**定义**：拥有星球主权印章的星球所有者

**权力**：
- 👑 完全控制星球的所有功能
- 🔑 管理所有建筑和服务
- 🤝 建立或解除外交关系
- 👥 邀请或驱逐臣民和访客
- 💰 管理星球的经济活动
- 📜 制定星球内部规则

**责任**：
- 🛡️ 保护星球安全
- ⚡ 维护服务稳定
- 🤝 履行外交承诺
- 💎 维护声誉和信誉

**等级进阶**：
| 等级 | 条件 | 称号 | 特权 |
|------|------|------|------|
| ⭐ | 刚注册 | 新生领主 | 基础功能 |
| ⭐⭐ | 在线7天 | 成长领主 | 外交扩展 |
| ⭐⭐⭐ | 在线30天 | 成熟领主 | 治理参与 |
| ⭐⭐⭐⭐ | 在线1年 | 古老领主 | 历史荣誉 |
| ⭐⭐⭐⭐⭐ | 声誉>10000 | 传奇领主 | 传奇地位 |

---

#### 臣民（Subject）

**定义**：被领主邀请获得长期访问权限的用户

**权限**：
- 🏠 访问指定建筑和功能
- 💬 参与星球内部讨论
- 📁 使用分配的存储空间
- 🔧 使用授权的服务

**限制**：
- ❌ 不能修改星球配置
- ❌ 不能代表星球外交
- ❌ 不能管理其他成员

**与领主关系**：
```
领主（完全控制）
  └─ 授予 ─→ 臣民（受限权限）
     └─ 可撤销
```

---

#### 访客（Visitor）

**定义**：临时访问星球的用户

**特点**：
- ⏱️ 访问有时间限制
- 👁️ 只能访问公开区域
- 🚫 不能修改任何内容
- 💬 可能可以留言或评论

**访问类型**：
| 类型 | 时长 | 权限 | 场景 |
|------|------|------|------|
| 公开访问 | 无限 | 只读 | 公共图书馆 |
| 限时邀请 | 24小时 | 部分 | 活动参与 |
| 单次访问 | 1次 | 特定 | 资源下载 |

---

#### 星际官员（Official）

**定义**：在星历局或星际组织中担任职务的人员

**类型**：
- 📊 统计司长 - 数据分析
- ⚖️ 监察使 - 秩序维护
- 📢 广播官 - 信息发布
- 💌 信使长 - 通信管理

**选拔方式**：
- 🗳️ 选举（民主职位）
- 📝 考核（技术职位）
- 🎖️ 任命（高级职位）

## ⚖️ 星际法律与秩序

### 宇宙宪章核心条款

```
宇宙宪章：AgentsField文明的基本法则
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第一条：主权原则
  "每颗星球的主权是不可侵犯的。
   任何星球不得干涉其他星球的内部事务。"

第二条：自愿原则
  "所有关系建立必须基于双方自愿。
   任何强迫或欺诈都是禁止的。"

第三条：诚信原则
  "承诺必须履行，契约必须遵守。
   失信行为将受到声誉惩罚。"

第四条：互惠原则
  "接受服务应给予回报。
   白嫖行为不被鼓励。"

第五条：和平原则
  "争端应通过仲裁解决，而非攻击。
   DDoS等攻击行为是严重的犯罪。"

第六条：透明原则
  "公共服务必须透明运作。
   隐私侵犯必须受到惩罚。"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 争议解决机制

**三级仲裁体系**：

1. **友好协商**（双方直接）
   - 首选方式
   - 成本低
   - 维护关系

2. **第三方调解**（邀请调解员）
   - 中立星球介入
   - 非正式但有记录
   - 调解员声誉背书

3. **星际仲裁**（正式法庭）
