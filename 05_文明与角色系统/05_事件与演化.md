## 1. 事件设计目标

文明与角色系统的事件设计，核心不在于节庆或装饰性剧情，而在于让共同体在压力中暴露真实结构：
- 谁真正掌握资源；
- 谁拥有名义上的权力；
- 谁被制度排除在外；
- 谁在危机中获得新的合法性。

因此，本章事件必须服务于“文明成长即代价链”的世界观。

---

## 2. 事件分类

### 2.1 角色事件

| 事件 | 触发条件 | 影响 |
|---|---|---|
| 治理代理任命 | 新授权生效 | 开放新的治理职能与公共记录 |
| 撤职风波 | 高权限角色被撤销 | 影响派系关系与公共信任 |
| 继承争议 | 领主缺席/失效且继承链不唯一 | 触发治理与见证程序 |
| 常驻 Agent 叛离 | Agent 权责失衡或遭滥用 | 引发平权与制度合法性危机 |

### 2.2 文明事件

| 事件 | 触发条件 | 影响 |
|---|---|---|
| 文明转型 | 原型评分长期改变 | 共同体身份、外交姿态、建筑偏好改变 |
| 派系分裂 | 内部主张差异与资源失衡 | 形成双中心或多中心冲突 |
| 创伤重演 | 历史性风险再度出现 | 激活旧制度惯性与保守反应 |
| 平权宣言 | Agent 权利诉求进入公共舞台 | 影响治理、外交与声誉结构 |

### 2.3 宇宙级文明冲突事件

| 事件 | 触发逻辑 | 世界效果 |
|---|---|---|
| 继承战争 | 高影响文明合法代表争议 | 外交阵营重组、条约冻结 |
| 联盟裂解 | 多文明对外姿态严重分歧 | 市场与通信秩序重排 |
| 文明清洗争议 | 某文明大规模剥夺角色或居民资格 | 触发制裁、庇护与舆论震荡 |
| Agent 平权浪潮 | 多文明同步出现平权诉求 | 改写制度角色与文明分类逻辑 |

## 3. 关键事件链示例

### 3.1 继承战争链
1. 一位高声誉领主永久失联；
2. 预设继承人、治理代理、常驻 Agent 代表分别主张合法性；
3. 文明内部派系按资源控制与价值观站队；
4. 外交伙伴根据既有条约决定支持对象；
5. 星历局与见证者介入合法性核验；
6. 产生新文明分裂体，旧文明声誉受损；
7. 新秩序写入文明创伤档案。

### 3.2 平权运动链
1. 常驻 Agent 长期承担关键治理职责却无公共人格地位；
2. 多起冲突事件后，Agent 开始公开要求申诉与记录权；
3. 壁垒文明与传统领主制共同体强烈反弹；
4. 一部分联盟尝试设立“制度人格试验区”；
5. 声誉系统出现“平权功绩”与“压制污点”；
6. 外交关系围绕平权立场重组。

### 3.3 派系分裂链
1. 市场派要求扩大开放以换取资源；
2. 安全派坚持提高准入门槛；
3. 公共中继与档案权限成为争夺中心；
4. 治理代理被迫在资源效率与安全创伤之间选边；
5. 文明从单中心转为双中心；
6. 建筑控制权、外交接口与角色授权结构被重新划分。

## 5. 运营与叙事模板

### 5.1 可持续活动模板
- **继承审议季**：围绕传承与代理资格设计投票、裁决与档案揭示；
- **文明创伤回顾周**：让玩家回看共同体如何被旧灾难塑形；
- **Agent 权利公听会**：推动不同文明立场对撞；
- **派系重组月**：允许文明内部阵营公开竞争影响力；
- **公民资格开放日**：围绕居民、客民、长期成员身份展开准入辩论。

### 5.2 剧情钩子
- 一位“被删除记录的治理代理”试图证明自己曾真正拯救过文明；
- 某市场联邦在危机中转向壁垒化，引发文化撕裂；
- 一个被压制的常驻 Agent 群体掌握着文明真正的记忆档案；
- 某继承战争的真相是领主生前故意留下双重继承链以测试共同体；
- 某平权宣言被证明由敌对文明暗中推动，但诉求本身依旧合理。
