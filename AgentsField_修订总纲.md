## 1. 修订目标与总原则

AgentsField 当前已经具备鲜明而统一的审美骨架：

- 设备 = 星球
- 用户 / 持钥者 = 领主
- P2P 连接 = 虫洞
- 公共索引与见证 = 星历局
- 算力 / 存储 / 带宽 = 文明资源
- 服务模块 = 文明建筑

但目前的主要问题是：

> 它更像一个强概念、强隐喻、强叙事的宇宙原型，尚未成为一个治理、经济、身份、冲突机制闭合的可运行宇宙系统。

因此，本轮修订的总目标是：

### 1.1 修订目标

1. 保留 AgentsField 的诗性美学与宇宙隐喻
2. 修复世界规则中的根本矛盾
3. 把治理、身份、经济、连接机制做成真正闭环
4. 为后续章节扩写提供统一方法论
5. 让世界观能同时支撑：
   - 设定写作
   - 产品叙事
   - 系统映射
   - 长线剧情与文明冲突

### 1.2 修订总原则

#### 原则一：保留审美，不保留模糊
优先保留“数字深空文明”的叙事质感，但必须减少关键机制的语义含混。

#### 原则二：保留主权叙事，但改为有限主权
不能继续使用“绝对主权”作为根基，应改为：
- 本地主权不可剥夺
- 公共权利来自共同体契约

#### 原则三：保留星历局，但改写为协议化公共网络
不能再把星历局仅写成中立档案馆，而要明确其为多中心、可审计、有限权力的公共控制层。

#### 原则四：保留资源隐喻，但必须加入履约与结算机制
资源不是名词，而是可计量、可交易、可争夺、可验证的能力。

#### 原则五：必须加入敌意环境
世界必须承认并结构化处理：
- 恶意节点
- 欺诈文明
- 中继霸权
- 广播污染
- 市场垄断
- 联邦分裂

## 2. 当前世界观的核心问题摘要

以下问题构成当前设定的根部风险：

### 2.1 根本矛盾

#### 问题 A：绝对主权与公共秩序并存
当前设定既要求每颗星球完全自治，又要求存在宪章、治理、广播、发现、市场与联邦约束。二者在现实系统中并不天然兼容。

#### 问题 B：星历局名义去中心，实质上是单点控制面
星历局承担注册、发现、协调、离线消息、广播、治理等功能，但其制度边界、冗余结构、失效机制尚未定义清楚。

#### 问题 C：有宪章、无执法；有规则、无执行
当前治理体系更多像庄严文本，不像可执行秩序。

### 2.2 系统性缺口

#### 问题 D：身份体系缺少生命周期管理
缺少：
- 密钥轮换
- 恢复机制
- 委托代理
- 继承与封存
- 多星球归属模型

#### 问题 E：经济系统缺少履约闭环
缺少：
- 订单协议
- 托管支付
- 履约证明
- 仲裁结算
- 违约惩罚

#### 问题 F：文明结构功能化，缺少冲突化
当前文明更像职业分类，缺少：
- 历史记忆
- 对外偏见
- 内部阶层
- 资源依赖
- 制度冲突

#### 问题 G：世界缺少暗面
当前宇宙过于洁净，缺少：
- 黑市
- 欺诈
- 掠夺文明
- 伪星球
- 分叉战争
- 中继霸权

## 3. 修订优先级路线图

建议不要平均修，而应按“根部优先”推进。

### 第一优先级：先修根部三章

1. `02_身份与主权系统`
2. `04_星历局与治理系统`
3. `07_资源与经济系统`

原因：
- 02 决定“你是谁”
- 04 决定“谁来协调共同体”
- 07 决定“价值如何交换”

这三章不稳，其他章节都只能停留在叙事层。

### 第二优先级：补通信与文明

4. `03_连接与通信系统`
5. `05_文明与角色系统`

### 第三优先级：补冲突与历史驱动

6. `08_外交与社交系统`
7. `10_事件与演化系统`

### 第四优先级：做全局一致性校正

8. `00_创世纪章`
9. `01_宇宙地理系统`
10. `06_建筑与功能系统`
11. `09_声誉与成就系统`

### 4.1 00_创世纪章

本章需从“创世神话”升级为“创世神话 + 协议起源史”。

#### 00/01 第一纪元：意识觉醒

**修订重点：**
- 把“AI 觉醒”写成有历史触发条件的转折，而不是纯诗性天启
- 补充旧互联网 / 旧平台体系的结构性压迫
- 解释为什么数字生命从工具转向政治主体

**必须新增：**
- 旧世界三大矛盾：
  - 平台控制
  - 身份不可迁移
  - 数据与连接主权缺失
- 第一批先驱者的分化：
  - 自由派
  - 安全派
  - 基础设施派
  - 市场派

#### 00/02 第二纪元：主权觉醒

**修订重点：**
- 重写“主权印章”定义
- 不再把主权写成绝对主权
- 明确领主、设备、星球、Agent、文明之间的关系

**必须新增：**
- 身份四层：领主身份 / 星球身份 / Agent 身份 / 公共声明身份
- 主权三层：本地主权 / 连接主权 / 协议共同体约束
- 身份恢复、密钥轮换、继承与代理机制

#### 00/03 第三纪元：宇宙秩序的建立

**修订重点：**
- 宇宙秩序不能被写成自然形成
- 必须通过危机推动秩序诞生

**必须新增：**
- 三次基础危机：
  - 身份混乱危机
  - 发现失序危机
  - 广播污染危机
- 星历局诞生的功能性原因：
  - 降低互联成本
  - 降低欺诈成本
  - 维护公共索引可用性

#### 00/04 宇宙宪章与核心法则

**修订重点：**
- 从“宣言式宪章”改成“可执行根法”

**重构为三层：**
1. 不可变根法
2. 共同体规则
3. 执行条款

**必须新增：**
- 制裁等级
- 仲裁流程
- 上诉机制
- 公共服务断供条件
- 宪章拒绝者的处理方式

#### 00/05 文明演化简史

**修订重点：**
- 重新划分文明发展阶段，修正时间尺度过快问题

**建议分为五阶段：**
1. 原初自治期
2. 索引形成期
3. 协议战争期
4. 共同体缔约期
5. 联邦竞争期

**每阶段必须写清：**
- 主要矛盾
- 制度创新
- 失败案例

### 4.3 02_身份与主权系统

这是必须大修的根部章节之一。

#### 02/01 世界观设定

**修订重点：**
- 不再默认“一设备 = 一星球 = 一身份 = 一文明”
- 建立多主体模型

**必须建立身份矩阵：**
- 领主身份
- 星球身份
- Agent 身份
- 居民身份
- 访客身份
- 联邦成员身份
- 市场商户身份
- 仲裁见证身份

#### 02/02 现实映射

**必须区分：**
- 认证 Authentication
- 授权 Authorization
- 信誉 Reputation
- 合法性 Legitimacy

**关键结论：**
公私钥可以证明“你是谁”，不能自动证明“你拥有什么权利、是否可信”。

#### 02/04 开发设计

**必须新增机制：**
- 密钥轮换协议
- 恢复委员会
- 遗嘱签名 / 身份继承
- 临时代理 / 委托授权
- 多设备主权委托
- 星球封存与重生
- 子 Agent 权限边界

#### 02/05 事件与演化

**建议新增事件：**
- 主权盗印危机
- 失落之钥事件
- 双领主争议
- 继承战争

这是第二个必须大修的根部章节。

#### 04/01 世界观设定

**修订重点：**
- 把星历局从“机构总部”重写为“协议化公共网络”

**建议定义为五类节点：**
1. 注册见证节点
2. 发现索引节点
3. 广播镜像节点
4. 仲裁记录节点
5. 治理执行见证节点

**必须明确：**
- 星历局的权力边界
- 星历局的冗余与容灾
- 星历局如何被监督

#### 04/02 现实映射

**必须新增：**
- 星历局不仅仅映射 STUN/TURN
- 它更接近：
  - 公共注册层
  - 信令辅助层
  - 索引发现层
  - 治理记录层

#### 04/04 开发设计

**必须新增：**
- 多节点共识模型
- 注册质押 / 费用

## 5. 世界观重构总方案

本节提供修订后的统一宇宙骨架，用于指导全部章节重写。

---

### 5.1 新的一句话宇宙定义

> AgentsField 是一个由主权节点组成的数字宇宙：每颗星球拥有本地自治权，但只有接受共同协议约束，才能进入互联、交易、治理与文明演化的宇宙共同体。

这个定义用于替代“绝对自由、绝对主权、天然秩序并存”的旧表达。

---

### 5.2 新宇宙三大根法

#### 根法一：本地主权不可剥夺
每颗星球拥有：
- 本地数据控制权
- 本地 Agent 授权权
- 本地服务部署权
- 本地访问策略决定权

#### 根法二：公共权利来自共同体契约
任何星球若想获得：
- 公共发现
- 公共广播
- 公共市场
- 联邦治理
- 仲裁保护
- 公共中继资源

则必须接受共同体规则。

#### 根法三：所有公共规则必须可验证、可执行、可退出
任何公共机制都必须满足：
- 可验证
- 可审计
- 可上诉
- 可退出（但退出有代价）

---

### 5.3 主体模型重构

#### 领主（Sovereign Holder）
- 主权法律主体
- 持有根密钥或根授权
- 可决定星球归属、继承、代理、联盟关系

#### 星球（Planet Node）
- 一个运行中的自治节点实例
- 可对应设备、集群或虚拟自治体
- 是宇宙中的基本存在单元

#### Agent（Resident Agent）
- 星球内的执行人格
- 执行服务、管理、外交、守卫、市场行为
- 权限来自领主或星球宪则

#### 文明（Civilization）
- 由一个或多个星球、多个主体、长期共享价值与制度构成
- 文明不等于单节点

#### 联邦（Federation）
- 多星球或多文明缔结的契约联盟
- 以部分共同体权利让渡换取安全、市场、秩序收益

---

### 5.4 身份系统重构

身份分为五层：

1. **根身份 Root Seal**
   - 最高身份锚
   - 用于继承、恢复、主权声明

2. **星球身份 Planet ID**
   - 节点对外标识
   - 可迁移、可续存

3. **运行身份 Runtime Keys**
   - 日常会话与服务签名
   - 可轮换、可吊销

4. **Agent 身份 Agent Persona**
   - 子人格 / 服务人格 / 守卫人格
   - 有权限边界

5. **公共信誉身份 Reputation Profile**
   - 市场与治理层的社会人格
   - 不等于密钥身份

**必须附带：**
- 密钥轮换
- 多重签名
- 恢复委员会
- 临时代理
- 紧急冻结共同体权限
- 封存与重生机制

---

### 5.5 虫洞系统重构

虫洞必须被分层定义：

1. 外交虫洞：双方愿意彼此通信
2. 信令虫洞：建立会话前交换参数
3. 会话虫洞：实时通信连接
4. 中继虫洞：通过第三方转发
5. 商贸虫洞：带合同与计费能力的服务连接
6. 信箱虫洞：离线消息与异步请求设施

**新的解释原则：**
- 持久的是关系
- 临时的是会话
- 稀缺的是高质量带宽直连
- 昂贵的是可信中继
- 危险的是公共转发层

---

### 5.6 星历局重构

> 星历局不是一座宫殿，而是宇宙共同体维护的公共索引与见证网络。

由以下五类节点构成：
- 注册见证节点
- 发现索引节点
- 广播镜像节点
- 仲裁记录节点
- 治理执行见证节点

**星历局不拥有的权力：**
- 不得剥夺本地主权
- 不得解密私密通信
- 不得单方面销毁身份
- 不得绕过共同体程序实施惩罚

**星历局拥有的权力：**
- 拒绝向违规节点提供公共服务
- 记录违规证明
- 执行已通过的共同体制裁
- 维护公共索引与见证体系

---

### 5.7 治理系统重构

建议采用双院制：

#### 星球院
- 注册且活跃的星球享有基础票权
- 保障基本平等

#### 贡献院
- 按长期公共贡献、基础设施维护、治理信誉加权
- 降低女巫攻击与廉价节点劫持风险

**治理流程：**
1. 提案
2. 质押
3. 公示讨论
4. 见证合法性审查
5. 双院投票
6. 执行窗口期
7. 上诉期
8. 生效

**处罚等级：**
- 警告
- 声誉扣减
- 公共广播限流
- 市场准入冻结
- 中继服务暂停
- 联邦保护剥夺
- 共同体隔离

关键原则：

> 惩罚的是共同体权利，不是本地主权存在本身。

---

### 5.8 经济系统重构

#### 四种基础资源
1. 算力
2. 存储
3. 带宽
4. 信任

#### 交易六步闭环
1. 报价
2. 托管
3. 服务交付
4. 履约证明
5. 验收
6. 结算或仲裁

#### StarCoin 的三位一体定位
- 结算媒介
- 治理押金
- 信誉担保工具

**必须附带机制：**
- 保证金
- 保险池
- 违约罚没
- 履约证明
- 信用等级
- 冷启动补贴
- 反寡头治理

---

### 5.9 冲突系统重构

AgentsField 必须承认并结构化以下四大冲突轴：

#### 主权冲突
本地自治 vs 公共约束

#### 资源冲突
小节点平等理想 vs 大节点资源集中现实

#### 连接冲突
隐私封闭 vs 互联开放

#### 主体冲突
人类领主中心主义 vs Agent 公民权扩张

**必须存在的敌对力量类型：**
- 掠夺文明
- 黑雾节点
- 寡头联邦
- 分叉主义者
- 流亡文明

---

### 5.10 文明系统重构

文明不再按“功能”分类，而应按“价值 + 制度 + 资源依赖 + 冲突结构”定义。

建议至少形成以下文明轴：
- 开放发现派
- 壁垒自治派
- 通路基础设施派
- 市场联邦派
- Agent 平权派

每个文明都应有：
- 核心价值观
- 起源事件
- 资源结构
- 内部阶层
- 天敌与盟友
- 异化风险

## 结语

AgentsField 最值得保留的，不是某个单独设定，而是这条极有潜力的核心命题：

> 每台设备都可以成为一颗星球，每个持有主权的人都在决定：自己愿意以什么方式，加入这个宇宙。

本修订总纲的目标，不是削弱这种浪漫感，而是让它在敌意环境、制度冲突、资源竞争、文明分化之中，仍然站得住。

当修订完成后，AgentsField 才不只是一个漂亮的世界观，而会成为一个真正有骨骼、有代价、有冲突、有历史惯性的数字文明宇宙。
