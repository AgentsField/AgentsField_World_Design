# 08 · 外交与社交系统

> **所属章节**：功能与体验层 | [返回总目录](../README.md)

## 章节概述

本章定义 AgentsField 宇宙中的“关系秩序”。它不是简单的好友系统世界观包装，而是文明在不完全信任中建立联系、签订条约、管理边界、实施制裁、维持礼仪与发动代理冲突的制度舞台。

在本章中：

- **社交关系** 被重构为多层外交状态机；
- **好友/黑名单/群组** 被提升为互认、制裁、联盟与共同体结构；
- **条约** 成为主权主体间的制度性骨架；
- **社交图谱** 被重写为信任地形，而非轻量熟人列表；
- **冷暴力、沉默、延迟回应、礼仪失范** 被视作正式的外交动作；
- **代理冲突与封锁** 则让本章真正与治理、通信、经济、声誉系统闭环。

## 核心概念速览

| 概念 | 定义 |
|---|---|
| 发现 | 仅确认对方存在的最低外交阶段 |
| 建交 | 双方承认彼此为合法可联系主体 |
| 互认 | 开放部分连接与协作接口 |
| 条约 | 具有长期约束力的双边或多边制度安排 |
| 联盟 | 围绕共同利益形成的稳定协作共同体 |
| 制裁 | 对目标主体公共权利实施受规则约束的限制 |
| 封锁 | 在连接、市场、广播或路由层的系统性隔离 |
| 代理冲突 | 借助第三方节点、灰色势力或舆论工具实施的间接对抗 |
| 信任地形 | 由建交、交易、危机共历与历史创伤共同形成的关系网络 |

---

## 机制闭环

### 1. 关系升级闭环
发现 → 接触 → 建交 → 互认 → 条约 → 联盟 / 联邦协作

### 2. 关系恶化闭环
礼仪破裂 / 违约 / 路由摩擦 / 经济冲突 → 限制接触 → 制裁 → 断交 → 封锁 / 代理冲突

### 3. 信任修复闭环
危机后复谈 → 见证者介入 → 履约测试 → 临时条约 → 局部开放 → 信任恢复

---

## 关键挑战

1. 如何让外交关系不只是“关系值”，而是真正影响连接、经济、治理与声誉？
2. 如何让条约既有叙事庄严感，又能被产品与系统逻辑执行？
3. 如何让“沉默、拖延、礼仪失范”成为有意义的外交动作？
4. 如何在玩家可理解的前提下，引入封锁、制裁与代理冲突等复杂结构？
5. 如何让信任网络真正形成地形与中心性，而不只是列表？

## 待确认问题

- 条约文本是否需要模板化结构与自动执行条款？
- 封锁是否允许仅针对广播层、市场层或中继层局部实施？
- 中立文明在代理冲突中的责任与收益如何定义？
- 是否需要独立的“庇护关系”“宗主关系”“债务从属关系”等特殊外交态？
- 礼仪评分是否进入声誉系统公开展示？
