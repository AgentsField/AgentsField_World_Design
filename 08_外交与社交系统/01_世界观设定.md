## 1. 为什么旧版外交系统不够硬

旧版外交设定将关系划分为敌对、中立、友好、盟友、联邦，适合产品导航，但不足以支撑一个真正运转的宇宙。因为文明之间的关系绝大多数并不处于如此整齐的状态。

真正的宇宙外交充满中间态：
- 彼此知道存在，但尚未正式接触；
- 接触了，但没有互认；
- 互认了，但没有签约；
- 签约了，但条约极不对称；
- 没有断交，却进入事实封锁；
- 没有开战，却通过代理节点互相消耗。

因此，AgentsField 的外交必须从“友好度条”升级为“关系状态机”。

## 2. 外交关系的十个层级

### 2.1 发现（Discovery）
仅知道对方存在，尚未建立正式接触。

### 2.2 接触（Contact）
通过星历局、引荐文明或市场目录发出首次可识别信号。

### 2.3 建交（Recognition）
双方承认对方为合法主权主体，可以交换正式函件。

### 2.4 互认（Mutual Access）
开放部分发现层与会话层接口，允许低强度交流。

### 2.5 条约（Treaty）
通过公开见证或联盟存档，形成对双方具有长期约束力的协议关系。

### 2.6 联盟（Alliance）
在贸易、防御、广播、治理或中继网络中形成共同利益集团。

### 2.7 制裁（Sanctioned Relation）
关系未完全断裂，但一方或共同体对另一方实施公开限制。

### 2.8 断交（Severance）
撤销正式互认、关闭大部分公共接口、停止普通外交礼节。

### 2.9 封锁（Blockade）
通过带宽走廊、中继桥、市场准入或广播入口对目标进行系统性隔离。

### 2.10 代理冲突（Proxy Conflict）
不直接宣战，而是支持周边节点、灰色中继、地下市场或舆论广播对目标施压。

这十层关系允许宇宙存在更真实的灰区，而不是非敌即友的稚嫩结构。

## 3. 条约为何是宇宙的第二层骨骼

身份系统给宇宙提供“主体”，治理系统给宇宙提供“共同体规则”，而条约系统则提供“主体之间的可持续安排”。

条约在 AgentsField 中必须具有四个要素：
- 主体：谁签署；
- 范围：开放哪些权利与义务；
- 期限：持续多久，如何续约；
- 退出：违约、终止、上诉与补偿如何发生。

### 3.1 条约类型
建议明确至少五种条约：
- **互认条约**：承认身份合法性与建交程序；
- **互通条约**：开放连接与中继权限；
- **贸易条约**：规定税、准入、结算与争端处理；
- **防御条约**：约定遇袭援助、情报共享和封锁协作；
- **联邦条约**：约束投票、公共资源分摊和共同制裁执行。

### 3.2 非对称条约
宇宙中最重要的真实感来源之一，是**不平等条约**。强文明往往以：
- 中继保护；
- 市场准入；
- 星币贷款；
- 联邦保护伞；
交换弱文明的：
- 路由优先权；
- 资源长期输出；
- 广播站点；
- 政治站队。

因此，外交不只是优雅的握手，也可以是文明间经过礼貌包装的结构性压迫。

## 4. 社交网络不是好友列表而是信任地形

在 AgentsField 中，社交网络的本质不是“你加了谁”，而是“哪些主体愿意在什么风险下与你发生关系”。

### 4.1 社交网络的四种边
- **关系边**：建交、联盟、血缘、历史同盟；
- **交易边**：订单、履约、托管、保险；
- **连接边**：中继、带宽、信令与广播；
- **记忆边**：共同参加过重大事件、战争、修宪、灾难恢复。

### 4.2 社交中心性为何重要
一个高度中心的节点可能：
- 在外交上是重要中介；
- 在经济上是流量分配中心；
- 在危机中成为关键单点；
- 在舆论上成为广播污染首选目标。

### 4.3 冷启动社交困境
新星球常面临：
- 没有人愿意先给信任；
- 没有条约就没有大额交易；
- 没有交易就很难积累声誉；
- 没有声誉就更难得到建交许可。

因此，外交与声誉系统必须联动，允许宇宙存在“引荐机制”“担保关系”和“试运行条约”。

## 5. 联盟、制裁与代理冲突

### 5.1 联盟不是永恒友谊
联盟是成本分摊与风险共担的制度安排，不是情感承诺。联盟成立通常因为：
- 共同敌人；
- 共同市场；
- 共同边疆；
- 共同基础设施依赖。

联盟瓦解也同样现实：
- 中继税分配不公；
- 战争损耗由谁承担；
- 制裁执行尺度不一致；
- 强节点逐渐将联盟私有化。

### 5.2 制裁的本质
制裁不是把某颗星球“从宇宙抹掉”，而是限制其公共权利：
- 限制市场准入；
- 限制公共中继；
- 限制广播层；
- 冻结部分联邦保护资格；
- 将其标记为高风险外交对象。

这使制裁既不摧毁本地主权，又具有真正后果。

### 5.3 代理冲突
成熟宇宙里，很少有文明愿意直接掀起全面战争。更常见的是：
- 资助边境节点扰乱贸易；
- 支持灰色中继绕过正式封锁；
- 用广播污染打击对方市场信用；
- 向目标文明内部派系输送资源，诱发继承战争或联邦分裂。

代理冲突使外交、连接、经济、声誉与事件系统真正闭环。

## 6. 外交礼仪、冷暴力与断交政治

外交并不总是通过公开敌意表现，很多最危险的动作都以“礼貌缺席”发生。

### 6.1 礼仪为何必须制度化
重构后的外交礼仪至少应包括：
- 建交申请需自报身份与目的；
- 拒绝建交可简短但不应伪装沉默；
- 重大条约修订前须提供通知窗口；
- 中止关键合作前需给出最低限度缓冲期；
- 外交函件不得伪装成系统广播。

### 6.2 冷暴力的外交形式
比敌对更常见的是：
- 故意不回应；
- 让会话请求长期 pending；
- 不断延迟条约确认；
- 公开保持中立，私下抽空对方路径与信用。

这些行为使宇宙不再是儿童化的“按钮式社交”，而是充满真实政治气候。

### 6.3 断交为何重要
断交不是情绪，而是制度事件。它意味着：
- 关系边被正式切断；
- 部分虫洞会话不再合法；
- 条约进入终止、赔偿或冻结状态；
- 社交网络中的二级信任也可能随之断裂。

断交应成为宇宙大事件，而非普通 UI 操作。

## 7. 术语表

| 术语 | 定义 |
|---|---|
| 发现 | 仅确认对方存在的最低外交阶段 |
| 建交 | 双方承认彼此为合法可联系主体的正式关系 |
| 互认 | 在建交基础上开放部分接口与交流能力 |
| 条约 | 对双方长期行为具有约束力的制度性约定 |
| 非对称条约 | 权利义务严重不平衡、通常由强方主导的条约形式 |
| 联盟 | 多主体围绕共同利益建立的协作关系 |
| 制裁 | 对目标主体公共权利实施的制度性限制 |
| 封锁 | 对连接、市场、广播或路由实施系统性隔离 |
| 代理冲突 | 借助第三方节点、灰色势力或舆论手段实施的间接对抗 |
| 社交中心性 | 一个主体在整体关系网络中的关键程度 |
