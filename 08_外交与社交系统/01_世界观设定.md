## 🤝 外交体系

### 外交关系类型

| 关系等级 | 标志 | 权限 | 建立条件 |
|---------|------|------|---------|
| **敌对** | ⚔️ | 拒绝连接 | 冲突升级 |
| **中立** | ⚪ | 基础通信 | 默认状态 |
| **友好** | 🤝 | 虫洞连接 | 双方同意 |
| **盟友** | 🛡️ | 资源共享 | 深度信任 |
| **联邦** | 🌐 | 共同治理 | 组织加入 |

### 建交流程

```
建交仪式：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 探测（Discovery）
   → 通过星历局发现目标星球
   → 查看对方档案和声誉

2. 申请（Application）
   → 发送建交请求
   → 附上自我介绍和意图

3. 审核（Review）
   → 对方评估申请
   → 查看申请者档案

4. 确认（Confirmation）
   → 接受或拒绝建交
   → 建立虫洞连接

5. 维护（Maintenance）
   → 定期互动维护关系
   → 互评和反馈

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 🌐 社交网络

### 社交图谱

```
星际社交网络：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

个人网络：
  直接联系（一度）：已建交星球
  扩展网络（二度）：盟友的盟友
  广泛网络（三度+）：宇宙中的可达星球

网络指标：
  - 网络密度：实际连接/可能连接
  - 中心性：在网络中的重要性
  - 聚类系数：朋友之间也是朋友的程度

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 🎩 星际礼仪

### 基本礼仪

1. **建交礼仪**：
   - 自我介绍清晰
   - 说明建交目的
   - 尊重对方回复时间

2. **通信礼仪**：
   - 及时回复消息
   - 语言文明礼貌
   - 尊重隐私边界

3. **贸易礼仪**：
   - 明码标价
   - 按时履约
   - 质量保证
