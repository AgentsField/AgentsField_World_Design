# 03 · 连接与通信系统

> **所属章节**：基础设施层 | [返回总目录](../README.md)

## 章节概述

本章定义 AgentsField 宇宙中的“连接秩序”。如果说 **02_身份与主权系统** 回答的是“你是谁、谁有权代表你”，那么本章回答的是“你如何被发现、如何建立接触、如何在不完全信任中交换信息，以及当连接失败时文明如何继续存在”。

在重构后的设定中：

- **P2P 连接** 不再被简单浪漫化为“天然虫洞”；
- **虫洞** 被拆解为发现、信令、会话、中继、离线信箱等多层机制；
- **连接主权** 被明确为本地主体对开放、拒绝、限流、隔离和中止连接的决定权；
- **公共连接秩序** 则由星历局公共索引网络、联盟目录、公共中继规范与广播边界共同维持；
- **敌意环境** 被正式写入世界：广播污染、幽灵星球、路由霸权、连接洪泛、离线伪函件、流量画像与通信战争不再是边缘题材，而是基础设定的一部分。

本章与以下章节形成强关联：

- **02_身份与主权系统**：决定谁能代表一个主体发起或接受连接；
- **04_星历局与治理系统**：提供公共索引、见证与连接秩序边界；
- **01_宇宙地理系统**：定义连接路径所处的空间层与可达性条件；
- **07_资源与经济系统**：决定中继、带宽、离线投递与连接税如何结算；
- **08_外交与社交系统**：决定连接关系如何升级为建交、条约、联盟或封锁；
- **10_事件与演化系统**：定义广播污染、断联时代、路由危机等系统性冲击如何发生。

## 核心概念速览

| 概念 | 定义 | 与旧设定的差异 |
|---|---|---|
| 建交关系 | 主体间的政治互认与可接触资格 | 不再等同于实时通信通道 |
| 发现信道 | 获取目标主体公共连接信息的索引层 | 不再默认开放全部内部结构 |
| 信令通道 | 协商路径、能力与会话参数的过程层 | 与实际载荷传输分离 |
| 会话虫洞 | 一次实际发生的端到端加密会话 | 从“永久神秘通道”改为短生命周期连接 |
| 中继桥 | 无法直连时的第三方转发路径 | 明确引入成本、依赖与路由霸权风险 |
| 离线信箱 | 实时连接失败时的延迟投递与回执设施 | 为断联文明提供连续性，而非临时补丁 |
| 连接主权 | 主体对开放、拒绝、限流、隔离和中止连接的权力 | 从抽象自由改为可执行边界 |
| 广播污染 | 对公共发现层与广播层实施的信息污染与钓鱼渗透 | 被提升为核心敌对环境 |

## 机制闭环

### 1. 连接建立闭环
发现 → 验证身份 → 协商参数 → 路径探测 → 会话建立 → 质量监测 → 正常结束 / 异常恢复

### 2. 连接失败闭环
失败探测 → 重试策略 → 切换候选路径 → 请求中继桥 → 转入离线信箱 → 记录信誉与风控结果

### 3. 公共秩序闭环
公共索引开放 → 陌生请求进入 → 主体执行本地策略 → 若形成骚扰/污染则触发限流、封锁、制裁或联盟协作治理

### 4. 经济闭环
中继与带宽服务提供 → 连接质量与成本产生结算 → 信用/声誉变化 → 影响后续可达性与价格 → 反向塑造连接结构

## 关键挑战

1. **如何维持诗意表达，同时避免将所有连接都浪漫化为天然平等？**
2. **如何在不破坏本地主权的前提下，为公共发现层建立反骚扰与反污染规则？**
3. **如何让中继桥既是基础设施，也是会带来政治与经济后果的权力节点？**
4. **如何让离线信箱成为制度化基础设施，而不是“掉线时补救一下”的附属机制？**
5. **如何让连接质量、带宽、延迟与中继成本，真正反馈到外交、经济与声誉系统中？**
