## 1. 事件设计目标

连接系统的事件设计，不应停留在“连接中断”“线路升级”这种局部效果，而应成为推动整个宇宙结构变化的重要引擎。通信层事件必须能够：

- 改变外交关系；
- 扰动市场与带宽价格；
- 触发治理回应与制裁程序；
- 在声誉层留下长期痕迹；
- 为 10_事件与演化系统 提供基础设施危机素材。

## 2. 事件分类

### 2.1 即时触发事件

| 事件 | 触发条件 | 表现 | 影响 |
|---|---|---|---|
| 首次成功建交会话 | 双方完成发现、验证与会话建立 | 建交通信仪式 | 提升互信并开放后续条约路径 |
| 中继接管 | 直连失败后成功切换桥接路径 | 路径重构、成本提示 | 增加结算成本并记录依赖关系 |
| 签名异常 | 身份验证失败 | 验证警报 | 触发风控与可疑主体记录 |
| 广播污染预警 | 污染评分超过阈值 | 公共层噪波扩散 | 限流、告警、影响公信力 |
| 离线函件成功投递 | 对方暂不可达但信箱可用 | 封蜡函件归档 | 保持外交/交易连续性 |

### 2.2 周期事件

| 事件 | 周期 | 内容 |
|---|---|---|
| 路径体检 | 每小时 | 评估连接质量、切换候选路径 |
| 公共索引净化 | 每日 | 清理异常发现记录与伪入口 |
| 中继桥结算 | 每日 | 统计带宽、转发量、租约费用 |
| 广播层审计 | 每周 | 检查污染源、虚假公告与异常传播 |
| 联盟互通巡检 | 每周 | 评估联盟间基础设施可靠性 |

### 2.3 世界级事件

| 事件 | 触发逻辑 | 世界影响 |
|---|---|---|
| 广播污染潮 | 多区域公共广播同时被污染 | 引发公共恐慌、错误制裁与舆情失真 |
| 路由霸权危机 | 少数桥接文明控制过多关键路径 | 带宽涨价、联盟反垄断、外交对立升级 |
| 大断联时代 | 公共索引、中继与目录同时大面积失效 | 宇宙转入局部自治与离线协同状态 |
| 通信冷战 | 两大联盟互不封锁但长期设置路径摩擦 | 中立节点价值暴涨，灰色中继繁荣 |
| 中继叛逃事件 | 高信誉桥接节点转向敌对阵营 | 引发信任崩塌与多方追责 |

## 3. 关键事件链示例

### 3.1 广播污染事件链
1. 某高中心性节点广播层被伪装公告污染；
2. 周边文明错误接受伪建交信号；
3. 外交系统出现误判与错误邀请；
4. 市场出现虚假订单与信用异常；
5. 星历局公共索引网络发起净化令；
6. 多个联盟争论是否启用公共制裁；
7. 相关节点在声誉系统中留下“污染污点”。

### 3.2 路由霸权危机链
1. 少数中继桥掌握边疆主要通道；
2. 过境价格持续提升；
3. 小文明被迫签订不对称通路条约；
4. 反垄断联盟成立；
5. 灰色中继市场快速扩张；
6. 治理系统讨论公共中继池与价格上限；
7. 通信战争风险进入高位。

### 3.3 大断联时代链
1. 公共目录分裂、桥接节点被攻击、广播层不可用；
2. 各文明转入低可见与本地自治模式；
3. 离线信箱与可信引荐链成为唯一有效联系形式；
4. 市场结算延迟，联盟决策效率暴跌；
5. 新的“断联时代协议”被提出；
6. 宇宙在恢复连接后进入新的制度阶段。

## 5. 运营与叙事建议

### 5.1 可持续活动模板
- **净网行动周**：集中清理广播污染、鼓励举报伪入口；
- **桥梁博览会**：中继文明展示服务能力，推动价格竞争；
- **断联生存演习**：模拟公共层中断，测试离线协作能力；
- **低延迟竞速季**：以连接质量排名驱动技术与基建升级；
- **静默礼仪月**：强调断交通知、建交通函、回执礼仪等软规则。

### 5.2 戏剧性叙事钩子
- 一封离线函件在多年后才被签收；
- 一个高信誉中继桥被揭露长期出售流量画像；
- 某联盟借“净化广播层”为名排斥边缘文明；
- 某断联小星球因为长期静默被误判为幽灵星球；
- 一场大范围污染事件反而促成新的公共索引宪章诞生。
