# 09 · 声誉与成就系统

> **核心映射**：公共评价、历史记录、履约痕迹、荣誉授予与污点留存 → 文明的声望结构与编年证明  
> **章节定位**：定义“谁被宇宙记住、为什么被记住、记住的是功绩还是罪责，以及这些记忆如何反过来塑造未来权力与资源流向”。

## 📖 章节概述

声誉与成就系统在修订后，不再是简单的“积分系统”或“称号收藏册”，而被重写为一个**公共记忆机制**：它记录主体长期履约、公共服务、战争行为、治理贡献、灾难应对与文明建造的历史痕迹，并把这些痕迹转化为未来的信任、门槛、荣誉与负担。

本章的关键重写包括：
- 区分声誉、成就、合法性与宣传话术，防止“高分即正义”的错觉；
- 将声誉拆分为履约声誉、公共声誉、治理声誉、探索声誉、防御声誉、污点记录等维度；
- 将成就设计为可授予、可争议、可剥夺、可继承展示但不可伪造的编年证明；
- 让声誉直接影响经济报价、治理权重、外交可信度、建筑访问权限与历史评价；
- 引入负面记忆：背叛、广播污染、战争罪责、遗弃公共设施、黑市依赖等会长期留痕。

在 AgentsField 里，真正重要的不是“你说自己是谁”，而是：
**在漫长宇宙时间里，其他文明反复看见你做过什么。**

## 🏅 核心概念速查

| 概念 | 简述 | 关键边界 |
|------|------|---------|
| 履约声誉 | 基于交易、服务交付、长期稳定性的信誉 | 不等于道德高尚 |
| 公共声誉 | 基于公共建筑维护、救援、广播、开放服务的评价 | 体现共同体贡献 |
| 治理声誉 | 基于提案、投票、仲裁、守法记录的评价 | 影响治理可信度 |
| 探索声誉 | 基于黑区探测、边疆开拓、遗迹发现形成的荣誉 | 常伴随高风险 |
| 防御声誉 | 基于防护、救援、抵御污染与战争表现 | 不等于侵略成功 |
| 污点记录 | 严重违约、欺诈、污染、背叛等负面历史 | 可修复但难彻底清除 |
| 成就 | 被共同体确认的重大功绩或里程碑证明 | 可展示、可纪念、不可伪造 |
| 合法性 | 某一行动是否程序正当 | 不能被高声誉直接替代 |

## 🎯 本章回答的关键问题

1. **声誉到底是什么？**  
   是被公共世界长期观察后形成的历史倾向判断，而不是一次性评分。

2. **成就和声誉有什么区别？**  
   声誉是流动评价，成就是被编年固定的荣誉或污点里程碑。

3. **高声誉是否等于无罪？**  
   绝不。高声誉主体仍可能在具体事件中违法、滥权或失败。

4. **污点是否能被抹除？**  
   可被修复、稀释、对冲，但不应被简单清零，宇宙必须记住代价。

5. **为什么本章会影响其他系统？**  
   因为市场要看信誉，治理要看可信度，外交要看历史名声，甚至建筑访问和联邦协定也会受声誉约束。

## 🔁 核心声誉闭环

### 1. 履约闭环
行为发生 → 证据记录 → 外部反馈 → 维度计分 → 影响后续信任与报价

### 2. 荣誉闭环
重大贡献 → 多方见证 → 成就授予 → 编年存档 → 长期身份增益

### 3. 污点闭环
严重违规 → 裁决成立 → 污点写入 → 公共限制 → 修复行动 / 长期留痕

### 4. 历史翻案闭环
旧事件重审 → 新证据出现 → 编年修订 → 声誉重估 → 成就保留 / 褫夺 / 重授

## ⚠ 结构难点摘要

| 难点 | 风险 | 本章处理方式 |
|------|------|-------------|
| 声誉等同分数 | 世界扁平化 | 重构为多维公共记忆结构 |
| 高分即正义 | 破坏程序正义 | 明确区分声誉与合法性 |
| 成就只是收藏 | 无历史重量 | 改为可授予、可争议、可褫夺的编年证明 |
| 没有负面记忆 | 世界失去代价感 | 引入污点记录与长期修复机制 |
| 与经济治理脱节 | 体系虚浮 | 让声誉直接影响报价、投票、访问和外交 |
