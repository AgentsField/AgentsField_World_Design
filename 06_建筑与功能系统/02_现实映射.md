## 🔗 服务模块技术映射

### 核心建筑技术对应

| 世界观建筑 | 技术实现 | 部署方式 | 资源需求 | 对外接口 |
|-----------|---------|---------|---------|---------|
| **家园核心** | 身份服务 + 配置管理 | 系统服务 | 基础资源 | REST API |
| **中继站** | TURN服务器 + 信令服务 | Docker容器 | 高带宽 | STUN/TURN |
| **知识宫殿** | 文件服务 + 对象存储 | 容器 + 存储 | 大容量 | HTTP/WebDAV |
| **交易所** | 撮合引擎 + 订单簿 | 微服务 | 高计算 | WebSocket |
| **酒馆** | 聊天服务 + 信令服务 | 容器集群 | 高并发 | WebSocket |
| **工坊** | 代码托管 + CI/CD | 容器 + 存储 | 高计算 | REST API |
| **娱乐中心** | 游戏服务器 + 媒体流 | 容器 + GPU | 高计算/带宽 | UDP/RTC |
| **防御要塞** | WAF + IDS + 审计 | 系统服务 | 高带宽 | 内部服务 |

## 🚀 建筑部署技术实现

### 部署流程

```
建筑部署流程：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 准备阶段
   - 检查资源充足性
   - 下载建筑镜像
   - 验证数字签名

2. 配置阶段
   - 生成配置文件
   - 分配资源配额
   - 设置网络参数

3. 启动阶段
   - 创建容器/进程
   - 初始化数据目录
   - 启动服务进程

4. 注册阶段
   - 注册到服务目录
   - 更新星历局档案
   - 启用对外访问

5. 验证阶段
   - 健康检查
   - 功能测试
   - 监控接入

技术实现：
  // 部署命令示例
  def deploy_building(building_type, level, config):
    // 1. 资源检查
    if not check_resource_availability(building_type, level):
      return Error("资源不足")
    
    // 2. 拉取镜像
    image = f"agentsfield/{building_type}:latest"
    pull_image(image)
    
    // 3. 创建容器
    container = create_container(
      image=image,
      resources=get_resource_limits(building_type, level),
      volumes=[f"/data/{building_type}:/data"],
      ports=get_exposed_ports(building_type),
      env=generate_config(building_type, config)
    )
    
    // 4. 启动服务
    start_container(container)
    
    // 5. 注册服务
    register_service(
      planet_opid=current_planet.opid,
      service_type=building_type,
      endpoint=get_service_endpoint(container),
      metadata=get_service_metadata(building_type, level)
    )
    
    return Success(container)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 升级资源成本

```
升级成本计算公式：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

升级成本 = 基础成本 × (1.5 ^ (当前等级 - 1))

示例：知识宫殿从Lv1升级到Lv2
  基础算力成本 = 10单位
  基础存储成本 = 100GB
  
  Lv1→2成本:
    算力 = 10 × (1.5^0) = 10单位
    存储 = 100GB
    时间 = 1小时
  
  Lv2→3成本:
    算力 = 10 × (1.5^1) = 15单位
    存储 = 200GB
    时间 = 2小时

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
