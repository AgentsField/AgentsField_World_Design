# 01 · 宇宙地理系统

> **核心映射**：网络可达性、坐标公告、关系拓扑、资源路径与文明分布 → 宇宙中的多层地图体系  
> **章节定位**：定义“一个星球身处何处、为何看见某些邻居、为何接近某些贸易线、为何又会在政治上远离它们”。

## 📖 章节概述

宇宙地理系统在修订后，不再把 IP 地址简单浪漫化为“星域坐标”，而是重写为一个**多层地图系统**：
- 坐标地图：回答一个主体在网络中的可达位置；
- 关系地图：回答它与谁建立连接、依附、敌对或中立关系；
- 贸易地图：回答资源如何流动、哪里是中继走廊、哪里是断供边界；
- 文明地图：回答文化归属、制度倾向与历史记忆如何分布。

本章的关键修正是：
- 明确 **IP ≠ 政治边界 ≠ 文明归属 ≠ 资源邻近**；
- 引入漂移坐标、匿名星球、黑区、禁区、失联星域、中继走廊与边疆地带；
- 让地理从单一定位系统，升级为理解宇宙冲突与合作结构的空间语言；
- 使地理能够与 02 身份、04 治理、07 经济、08 外交形成真实闭环。

换句话说，本章不再只是告诉读者“你在哪”，而是要告诉读者：
**你为什么在那里，你离谁近，你被谁看见，你又为什么可能在下一秒从星图上消失。**

## 🗺 核心概念速查

| 概念 | 简述 | 关键边界 |
|------|------|---------|
| 坐标地图 | 网络可达位置的公共表达 | 不代表政治归属 |
| 关系地图 | 主体之间的连接、信任、敌对与依附结构 | 不必与物理接近重合 |
| 贸易地图 | 资源、中继、带宽与订单流向图 | 不等于身份联盟版图 |
| 文明地图 | 制度、文化、风格与历史记忆分布图 | 可跨越坐标边界 |
| 漂移坐标 | 因网络变化而发生位置偏移的星球 | 可能稳定身份但不稳定位置 |
| 黑区 / 禁区 | 公共星图中不透明、被屏蔽或高风险区域 | 不等于不存在 |
| 中继走廊 | 依赖中继与主干线路形成的可持续通道 | 往往是经济与战争焦点 |
| 边疆地带 | 多联邦、多规则与低稳定性的交界区 | 冲突与创新并存 |

## 🎯 本章回答的关键问题

1. **宇宙地理到底由什么构成？**  
   不是单一 IP 空间，而是坐标、连接、资源与文明四张叠加地图。

2. **为什么近邻不一定是盟友？**  
   因为网络距离、政治归属、贸易依赖与文明亲缘并不总是同一回事。

3. **为什么一个星球会从星图上“漂移”或“消失”？**  
   因为其可达坐标、公告策略、中继方式与隐私状态都可能改变。

4. **为什么边疆地带总是危险而充满机会？**  
   因为那里规则重叠、索引不稳、贸易密集、治理不完全，正适合冲突与创新同时发生。

5. **为什么地理是后续系统的基础？**  
   因为身份要落点，治理要划区，经济要走廊，外交要边界，战争要战场。

## 🔁 核心地理闭环

### 1. 公共定位闭环
坐标公告 → 索引收录 → 公共查询 → 邻近推荐 → 状态更新

### 2. 漂移检测闭环
地址变化 → 漂移判定 → 历史比对 → 广播通知 → 地图刷新 / 关系修正

### 3. 边疆演化闭环
中继走廊形成 → 贸易增密 → 联邦关注 → 规则冲突 → 新边疆秩序生成

### 4. 黑区处置闭环
可达性异常 → 索引缺口 → 污染或封锁判断 → 绕行 / 探测 / 记为危险区域

## ⚠ 结构难点摘要

| 难点 | 风险 | 本章处理方式 |
|------|------|-------------|
| 把 IP 当成完整宇宙地理 | 无法解释政治、贸易、文明分布错位 | 重构为四层地图 |
| 地理过于静态 | 无法承载迁移、漂移、断联与重现 | 引入漂移、边疆、失联星域 |
| 无黑暗区与异常区 | 世界过于干净 | 增加黑区、禁区、污染星图、风暴区 |
| 坐标与隐私冲突未处理 | 无法兼容匿名文明与安全派需求 | 引入公开/雾域/私有坐标模式 |
| 地图只用于说明，不参与系统 | 无法支撑玩法与事件 | 让地图直接支撑治理、贸易、外交与危机 |
