## 🔗 技术→世界观映射表

| 技术概念 | 世界观表现 | 映射逻辑 | 用户感知 |
|---------|-----------|---------|---------|
| IP地址 | 星球坐标 | 数字→宇宙位置 | 看到星图定位 |
| IPv4第一段 | 星域 | 1-255→四大星域 | 所属大区 |
| IPv4第二段 | 星系 | 0-255→星系编号 | 所属星系 |
| 完整IP | 星球 | 唯一地址→唯一星球 | 精确定位 |
| 公网IP | 现实空间 | 可直接访问 | 清晰可见 |
| NAT内网 | 迷雾空间 | 需穿透技术 | 模糊不清 |
| 离线状态 | 虚空空间 | 不可访问 | 完全隐藏 |
| 动态IP | 坐标漂移 | IP变化 | 位置移动 |

## 🌐 地址示例

### IPv4映射

```
IP: 203.0.113.42
→ 星域: 南域（203）
→ 星系: 0号星系
→ 星球: 113.42号星球
```

### IPv6映射

```
IP: 2001:0db8:85a3:0000:0000:8a2e:0370:7334
→ 星域: 北极域（2001）
→ 星系: 0db8号星系
→ 恒星系: 85a3号恒星系
→ 星球: 完整128位地址
```

## 🌐 IPv6完整映射规则

| IPv6部分 | 世界观表现 | 映射逻辑 | 示例 |
|---------|-----------|---------|------|
| 前16位 | 星域 | 2000-2700划分8域 | 2001→北极域 |
| 次16位 | 星系 | 0000-ffff | 0db8→3512号星系 |
| 第三组16位 | 恒星系 | 0000-ffff | 85a3→34211号恒星系 |
| 完整128位 | 星球唯一ID | 全地址 | 精确定位 |

### 双栈星球映射

```
双栈星球示例：
IPv4: 203.0.113.42 → 南域/0号星系
IPv6: 2400::1 → 南域（新宇宙）
显示: "南域双栖星球"
```

## 🔄 状态转换

| 技术变化 | 世界观表现 |
|---------|-----------|
| 获得公网IP | 从迷雾空间进入现实空间 |
| 进入NAT | 从现实空间进入迷雾空间 |
| 断网 | 沉入虚空空间 |
| 重新上线 | 从虚空空间复苏 |
